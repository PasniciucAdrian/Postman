# 📌 Create New Board - Trello API

This request allows you to **create a new board** named `Postman-Trello` using the Trello API.

---

## 🔧 Method & Endpoint

**POST**  
`https://api.trello.com/1/boards/`

---

## 🧩 Request Body

```json
{
  "name": "Postman-Trello",
  "key": "{{apiKey}}",
  "token": "{{apiToken}}"
}
```

---

## ✅ Test Scripts (Postman)

// Extract and display the `id` and `name` of the created board
```javascript
pm.test("Status code is 200", function () {
    pm.response.to.have.status(200);
});

let res = pm.response.json();
console.log("Board ID:", res.id);
console.log("Board Name:", res.name);

pm.test("Board name is exactly 'Postman-Trello'", function () {
    pm.expect(res.name).to.eql("Postman-Trello");
});

pm.test("Board ID is present", function () {
    pm.expect(res.id).to.not.be.undefined;
});

pm.environment.set("boardId", res.id);
```
