# 📌 Get All Boards - Trello API

This request fetches **all boards** associated with your Trello account using the Trello API.

---

##  Method & Endpoint

**GET**  
`https://api.trello.com/1/members/me/boards?key={{apiKey}}&token={{apiToken}}`

---

## ✅ Test Scripts (Postman)

// Log and test that the desired board exists
```javascript
pm.test("Status code is 200", function () {
    pm.response.to.have.status(200);
});

let boards = pm.response.json();
console.log("All Boards:", boards);

let boardFound = boards.find(board => board.name === "Postman-Trello");

pm.test("Board 'Postman-Trello' is found", function () {
    pm.expect(boardFound).to.not.be.undefined;
    console.log("Board ID:", boardFound.id);
    pm.environment.set("boardId", boardFound.id);
});
```

---

##  Test Results

- **Status**: `200 OK`
- **Board Found**: `Postman-Trello`
