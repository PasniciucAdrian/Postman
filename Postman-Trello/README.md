\# 📋 Trello Workflow Simulation



This folder contains a sample Trello board created to demonstrate how a junior manual tester can organize tasks and test cases in a real project environment.



\## 🔍 Purpose



The board was built to simulate a software testing workflow using Trello as a task management tool. It contains:



\- To Do: test cases to be written

\- In Progress: tests currently being written or executed

\- Testing: test cases under active execution

\- Bugs: issues identified during manual testing

\- Done: completed test cases



\## 🛠️ Tools Used



\- Trello (task board)

\- Manual testing methodology
